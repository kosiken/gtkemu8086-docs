You can also add your own personalized theme by clicking add in the
fonts and colors section of the Preferences window.

Here is a template

```ini
[Theme]
keyword=#268BD2
reg=#B58900
string=#2AA198
label=#A6E22E
num=#d33682
special=#cb4b16
comment=#586E75
highlight=rgba(10,58,70,1)
background=rgba(3,47,59,1)
cursor=#ffffff
selection=#002B36
selectionbg=#657B83
text=#93A1A1
linecolor=#586E75
[Description]
value=Color scheme using Solarized dark color palette

```
You can save this in a file *new-theme.theme* and import it 

**et voila!!**
