## Linux

### Building from source

The source tarball can be gotten from [here](https://github.com/kosiken/gtkemu8086/archive/1.0.0.tar.gz)
The following are the dependencies of it
- gtk+>=3.0
- libxml
> You need to have the dependencies installed to compile program
```
./configure
make
make install

```

## Windows

### Using the insaller

<Emphasis>
  The windows app is a 64 bit application if you are looking to get one for another architecture you can follow the 
section <a href="#from-source">below</a>
  
  </Emphasis>
  


- Download the installer from this [link](https://github.com/kosiken/gtkemu8086/releases/download/1.0.0/emu8086setup.exe)
- Run the installer to install it on your computer

## From source 

- Clone the [repository](https://github.com/kosiken/gtkemu8086)
- Download msys2 you can find it [here](https://www.msys2.org/#installation)
- Then follow the official gtk+ [guide](https://www.gtk.org/docs/installations/windows)
