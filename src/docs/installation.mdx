## Linux

### Building from source

The source tarball can be gotten from [here](https://github.com/kosiken/gtkemu8086/archive/1.0.2-beta.tar.gz)
The following are the dependencies of if you are building from source

- gtk+>=3.0
- libxml
- libpeas 

> You need to have the dependencies installed to compile program

```sh

$ ./configure
$ make
$ make install 

```
