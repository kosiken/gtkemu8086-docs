## Linux

### Building from source

The source tarball can be gotten from [here](https://github.com/kosiken/gtkemu8086/archive/1.0.0.tar.gz)
The following are the dependencies of it
- gtk+>=3.0
- libxml
> You need to have the dependencies installed to compile program
```
./configure
make
make install

```
